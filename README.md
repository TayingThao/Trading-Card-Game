# Trading Card Game

By: Taying Thao

A web-based trading card game

### Running the Application

To start the application, navigate to the server package and run:

cd packages/server
npm run start:app

This will start both the backend server and the frontend application.

The application will be available at `http://localhost:3000` (or the configured port).

## Project Structure

- `packages/app/` - Frontend application
- `packages/server/` - Backend server
- `packages/proto/` - Prototype/static files
